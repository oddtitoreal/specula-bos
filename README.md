# Specula BOS

## Futures-Driven Brand Operating System

**Il prodotto centrale di Specula Future Crafting.**

Non un framework da attraversare una volta, ma un'architettura cognitiva viva che collega futuri, identità, decisioni e governance in un sistema capace di apprendere.

> *I futuri non si predicono. Si abitano. Il brand che ha attraversato il BOS non sa cosa succederà: sa chi è abbastanza bene da poter rispondere a ciò che succede senza smettere di essere se stesso.*

---

## Cosa contiene questo repo

Questo repository documenta la parte **aperta** del BOS: la struttura architettonica, i workshop, il layer etico, il Circolo di Sintesi, la Memory Architecture, i modelli di ingaggio, e la Definition of Done.

### Aperto (questo repo)
- Architettura dei 7 layer (WORLD → FUTURES → IDENTITY → EXPLORATION → STRATEGY → GOVERNANCE → INTERFACE)
- Ethical Frame trasversale (Immutable Values · Ethical Gate · Refusal Register)
- Circolo di Sintesi dell'Identità — composizione, mandato, ritualità
- Memory Architecture — 5 domini (Scenario, Brand, Decision, Refusal, Context)
- Workshop protocols per ogni layer
- Brand Governance Playbook — struttura dei 6 strati
- Modelli di ingaggio (Audit · Full Journey · Guardian)
- Definition of Done

### Proprietario (non in questo repo)
- Configurazioni AI e librerie di prompt
- Librerie di scenari e protocolli di ricerca
- Strumenti operativi di facilitazione
- Memory Architecture e schemi JSON (implementazione)
- Lavoro specifico su clienti e case study
