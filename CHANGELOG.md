# Changelog

All notable changes to the Specula BOS documentation will be documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [2.0.0] - 2026-06-04

### Added
- Initial public release of the BOS architectural documentation
- `docs/` — 8 documents covering all parts of the BOS (diagnosis, principle, 7 layers, Circolo di Sintesi, Memory Architecture, system lifecycle, open ethical layer, Definition of Done)
- `workshops/` — 10 workshop protocols, one per layer plus cross-cutting workshops
- `templates/brand-governance-playbook.md` — open template for the 6-layer governance playbook
- `README.md` with ecosystem positioning and open/proprietary split
- `docs/00_index.md` — documentation map for different audiences

### Notes
- This is the first public release of the BOS architectural docs
- Framework reference: Specula v2.4.5
